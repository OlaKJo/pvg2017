Användarmanual Enduro Race System

Introduktion

Detta system är framtaget för Enduro Race och används för att samla in tider samt beräkna resultat. Uppe till vänster ser du alla rubriker. Klicka på en rubrik för att gå till önskat stycke.

Installation

Systemkrav

  • En dator
  • Java version 8
  • Operativ system som kan köra Java 8, se https://www.java.com/en/download/help/sysreq.xml

Föranmälan av deltagare

Textfil för föranmälan av deltare är viktig. Denna fil måste placeras i rätt fil enligt er konfiguration.

Maratonlopp

Vid ett marathonlopp finns det en startpunkt och en slutpunkt. Registrera tider för båda punkterna för alla deltagare.

Registrering

Kör registreringsprogrammet genom terminalen: java -jar registration-1.0.jar

Enkel start

  • Specificera att det är en startstation genom att använda drop-down-menyn Station Type och välja Start.
  • Välj Station ID till valfritt unikt stationsnummer.
  • Klicka sedan på Start Registration för att påbörja registreringen.
  • Fyll i Start Number med en deltagares unika startnummer.
  • Klicka på Register för att registrera deltagaren vid denna tidpunkt.
  • Vid lyckad registrering kommer den registrerade tiden synas i listan utav de fem senaste registreringarna.

Masstart

  • Specificera att det är en startstation genom att använda drop-down-menyn Station Type och välja Start.
  • Välj Station ID till valfritt unikt stationsnummer.
  • Tryck på knappen Masstart, detta ger alla deltagare samma starttid.

Slutregistrering

  • Specificera att det är en slutstation genom att använda drop-down-menyn Station Type och välja Finish (or lap time).
  • Välj Station ID till valfritt unikt stationsnummer.
  • Klicka sedan på Start Registration för att påbörja registreringen.
  • Fyll i Start Number med en deltagares unika startnummer.
  • Klicka på Register för att registrera deltagaren vid denna tidpunkt.
  • Vid lyckad registrering kommer den registrerade tiden synas i listan utav de fem senaste registreringarna.

      • Förregistrering

    • Om du behöver förregistrera, klicka på knappen register utan att fylla i något startnummer
    • Den förregistrerade tiden kommer nu synas under Pre-registrations
    • Fyll i startnummer för den förregistrerade tiden
    • Tryck på save för att spara tiden eller clear för att förkasta den förregisterade tiden
    • Obeservera, du kan endast spara undan en förregistrerad tid åt gången

Varvlopp

Varvloppet kan beskrivas som en sekvens av marathonlopp där den sista registrerade varvtiden representerar den slutgiltiga sluttiden.

Registrering

Kör registreringsprogrammet genom terminalen:

java -jar registration-1.0.jar

Enkel start

  • Specificera att det är en startstation genom att använda drop-down-menyn Station Type och välja Start.
  • Välj "Station ID" till valfritt unikt stationsnummer.
  • Klicka sedan på Start Registration för att påbörja registreringen.
  • Fyll i Start Number med en deltagares unika startnummer.
  • Klicka på Register för att registrera deltagaren vid denna tidpunkt.
  • Vid lyckad registrering kommer den registrerade tiden synas i listan utav de fem senaste registreringarna.

Masstart

  • Specificera att det är en startstation genom att använda drop-down-menyn Station Type och välja Start.
  • Välj "Station ID" till valfritt unikt stationsnummer.
  • Tryck på knappen Masstart, detta ger alla deltagare samma starttid

Varvtidregistrering och slutregistrering

  • Specificera att det är en slutstation (eller varvstation) genom att använda drop-down-menyn Station Type och välja Finish (or lap time).
  • Välj Station ID till valfritt unikt stationsnummer.
  • Klicka sedan på "Start Registration" för att påbörja registreringen.
  • Fyll i Start Number med en deltagares unika startnummer.
  • Klicka på Register för att registrera deltagaren vid denna tidpunkt.
  • Vid lyckad registrering kommer den registrerade tiden synas i listan utav de fem senaste registreringarna.
  • För att registrera varvtider använd dig av flera sluttider, endast den sista sluttiden kommer representera den faktiska sluttiden.

      Förregistrering

    • Om du behöver förregistrera, klicka på knappen Register utan att fylla i något startnummer
    • Den förregistrerade tiden kommer nu synas under Pre-registrations
    • Fyll i startnummer för den förregistrerade tiden
    • Tryck på save för att spara tiden eller clear för att förkasta den förregisterade tiden
    • Observera, du kan endast spara undan en förregistrerad tid åt gången

Resultatgenerering

    Konfiguration

    För att själv definiera var namnfil, tidsregistreringar och resultat ska hämtas respektive genereras kan man använda sig av en konfigurationsfil. Det finns en default-fil i releasen med namn: Config_DoNotMoveOrDeleteThis. Genom att antingen ändra direkt i denna fil eller att skapa en egen config-fil enligt samma format kan man ändra konfigurationen och skapa flera konfigurationer. Konfigurationsfilen anges som parameter i terminalen enligt nedan: > java -jar sorter.jar my_own_config_file Observera om ingen konfigurationsfil anges, används defaultvärden.

    Maratonlopp

    För att köra detta läge måste fältet raceType sättas till marathon i konfigurationsfilen.

    För att generera resultatfil körs sorterprogrammet via terminalen, java -jar sorter-1.0.jar. Detta skapar resultatfilerna. Om inget annat anges sparas dessa i samma directory som jar-filen körs ifrån. De namnges då:
    • result.txt,
    • resultsorted.txt,
    • result.html_sorted.html.
    Resultatfiler genereras i startnummerordning med totaltider
    Med format: Start nbr; Name; Total time; Start time; Finish time

    Varvlopp

    För att köra detta läge måste fältet raceType sättas till lap i konfigurationsfilen.

    För att generera resultatfil körs sorterprogrammet via terminalen, java -jar sorter-1.0.jar. Detta skapar resultatfilerna. Om inget annat anges sparas dessa i samma directory som jar-filen körs ifrån. De namnges då:
    • result.txt,
    • resultsorted.txt,
    • result.html_sorted.html.
    Resultatfiler genereras i startnummerordning med totaltider (och varvtider).
    Med format: Start nbr; Name; #Lap; Total time; Lap times; ... Start time; Lap; ... Finish time
